# Sublime Text

For the full settings schema, see [Setup Overview](/setup).

## Required package

- [LSP](https://github.com/sublimelsp/LSP) — Language Server Protocol client for Sublime Text

Install via Package Control: `Ctrl+Shift+P` → `Package Control: Install Package` → `LSP`.
