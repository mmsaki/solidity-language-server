## Using Generic LSP Proxy extension

For VS Code (and Cursor), you can run through the Generic LSP Proxy extension:

- Extension: [mjmorales.generic-lsp-proxy](https://marketplace.visualstudio.com/items?itemName=mjmorales.generic-lsp-proxy)
- This proxy config is **in addition to** the `solidity-language-server.*` settings below (not a replacement).

Config example:

```json [settings.json]
[
  {
    "languageId": "solidity",
    "command": "forge",
    "fileExtensions": [
      ".sol"
    ],
    "args": [
      "lsp"
    ]
  }
]
```
