## Generic LSP Proxy extension

Install the [Generic LSP Proxy](https://marketplace.visualstudio.com/items?itemName=mjmorales.generic-lsp-proxy) extension.

Add the following to `.vscode/lsp-proxy.json` in your project root:

```json [.vscode/lsp-proxy.json]
[
  {
    "languageId": "solidity",
    "command": "solidity-language-server",
    "fileExtensions": [
      ".sol"
    ]
  }
]
```

## Lint options reference

Lint options (severity, rule IDs, `only`, `exclude`) are documented in [Setup Overview → Lint values](/setup#lint-values).

Example filtered lint setup:

```json [settings.json]
{
  "solidity-language-server.lint.severity": ["high", "med"],
  "solidity-language-server.lint.exclude": ["pascal-case-struct"]
}
```
